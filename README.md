# VeloraVouchers

VeloraVouchers is a Paper plugin for creating, managing, and redeeming configurable voucher items on Minecraft servers.

It supports redeemable voucher items with command rewards, item rewards, chance-based outcomes, permission checks, placeholder conditions, GUI editing, and anti-duplication protection.

## Features

- Redeemable voucher items
- Command rewards with chance support
- Item rewards with chance support
- GUI-based voucher editor
- Voucher list GUI
- Stackable and non-stackable voucher modes
- Anti-duplication protection for non-stackable vouchers
- Placeholder-based redeem conditions
- Permission-based redeem restrictions
- Custom deny messages and actions
- Claim-all support
- Staff alerts for suspicious stacked vouchers

## Voucher Modes

### Stackable
- Better inventory usage
- Easier to distribute
- No unique item tracking
- Less secure against duplication abuse

### Non-Stackable
- Each voucher gets a unique UUID
- Full claim tracking
- Better protection against duplication exploits
- Recommended for secure voucher systems
